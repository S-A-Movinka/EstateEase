EstateEase - Property Search Web Application

Project Description:
A responsive React-based web application that allows users to search for properties based on various criteria such as price, location, and type. This project was developed for the Advanced Client-Side Web Development (5COSC026W) module.

Key Features

Advanced Search: Search by type, price range, bedrooms, date added, and postcode .


Property Details: Individual pages with image galleries and tabbed content (Description, Floor Plan, Maps) .


Favourites System: Drag-and-drop or button-based functionality to save and manage a list of properties .


Responsive Design: Optimized for both large screens and mobile devices.

Technical Stack

Frontend: React JS 


Data: Local JSON file 


Styling: Flexbox/Grid and Media Queries 


Testing: JEST 

How to Run
Clone the repository.

Run npm install to install dependencies.

Run npm start to view the app at http://localhost:3000.
